CL-RFC2047

Implmentation of RFC2047.

DECODE

Function

Decode encoded word in STRING from START to END.

DECODE*

Function

Decode mixed STRING (possibly containing encoded as well as unencoded words) from START to END, if ERRORP is non nil DECODE* will return the string unmodified if an error occurs.

ENCODE

Function

Return encoded STRING using ENCODING and CHARSET.

SHOULD-ENCODE-P

Function

Predicate to test if STRING should be encoded.
